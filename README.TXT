PROJECT TITLE: Juego Scape Room Medieval
PURPOSE OF PROJECT: Tarea Programada 
VERSION or DATE:1/10/2020
HOW TO START THIS PROJECT:Javac Main.java para compilar, java Main (consola o Paneles) para iniciar.
AUTHORS: Brenes Hernández Saindell Sabrina, Wu Nie Johana, Bonilla Rivera Gabriel. 


Instrucciones generales para programar:
Dar bienvenida 
pide el nombre 
pregunta lo que quiere hacer (jugar, salir, ver record)
puede tomar tres caminos, sale la ventana de record que es solo un mensaje y le vuelve a presentar las opciones iniciales, puede salir del juego, o puede jugar 
en el caso de jugar, le dice que ya está dentro de la primera habitación y le enseña sus stats, le pregunta si quiere mover, ver el registro o abandonar
en el caso de abandonar, le pregunta si quiere jugar otra vez
en el caso de ver el registro le presenta un mensaje con sus acciones hasta el momento y le vuelve a tirar las opciones in game 
en el caso de hacer un movimiento, le pregunta la carta, y le pregunta la cantidad de ataques 
Sucede la magia, se toman las cartas se realiza el cálculo y se refleja en la pantalla de Stats, en la clase registro y en los toString()
Se le vuelve a preguntar 
Así sucesivamente hasta que salga de la habitación o muera 
Cuando muere, se le enseña su resultado final y los récords nuevamente 
Se le pregunta si quiere volver a jugar 
Puede tomar dos caminos, jugar otra vez o salir del todo
